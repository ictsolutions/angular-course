# HTML5 Index

## What is HTML?

Hypertext Markup Language.  

Technologies that drive the web:  
- HTML -> Structure  
- CSS -> Style  
- Javascript -> Behaviour  

```html
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>My First Heading</h1>
        <p>My first paragraph.</p>
    </body>
</html>
```
